## 🚀 Features
- User login/logout
- Add, edit, and delete tasks
- Task fields: Title, Description, Priority, and Due Date
- Clean and responsive UI

---

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Supabase
- Responsive design with mobile-first approach
